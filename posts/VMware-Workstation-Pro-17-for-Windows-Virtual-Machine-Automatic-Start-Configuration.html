VMware Workstation Pro 17 for Windows 开机自启设置

发表于 2023-02-19 更新于 2023-06-25 分类于 VM 阅读次数： Waline： 阅读时长 ≈ 1 分钟

前言

VMware Workstation Pro 17.0 增添了“配置自动启动虚拟机”，但需要进行一些配置才能实现该功能，有些繁琐，可能会在17.1、17.2之类的版本中完善

参考

【老湿基】VMware Workstation Pro 17 开机自动启动虚拟机教程

设置方式

修改“VMware 自动启动服务”

当安装完成 VMware Workstation Pro 17.0 后，在 Windows 的服务中可以找到“VMware 自动启动服务”，显示的是“手动”

修改为如下，修改完成后再点击启动手动启动一次

常规——启动类型——自动 登录——此账户——浏览，添加你用的账户

添加自动启动的虚拟机

以管理员权限运行 VMware Workstation Pro，选择“文件——配置自动启动虚拟机”，勾选需要自启动的虚拟机
