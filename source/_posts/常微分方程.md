# 常微分方程（ODE）

## 解析解

常微分方程（ODE）最简单的形式是 $\frac{dy(x)}{dx}=f(x, y(x))$，其中 $y(x)$ 是未知函数，$f(x, y(x))$ 是已知的。

例一

一阶 ODE： 牛顿冷却定律，$\frac{dT(t)}{dt} = -k(T(t) - T_a)$，初值为 $T(0) = T_0$。求解步骤如下

0. 导入包

   ```python
   import numpy as np
   import matplotlib.pyplot as plt
   from scipy import integrate
   import sympy
   
   sympy.init_printing()  # 初始化其屏幕打印系统
   ```

1. 使用 `sympy.symbols` 定义符号 $t, k, T_0, T_a$ 、使用 `sympy.Function` 定义函数 $T(t)$

   ```python
   t, k, T0, Ta = sympy.symbols('t, k, T_0, T_a')  # 定义符号
   T = sympy.Function('T')  # 定义函数
   ```

2. 定义ode表达式（公式写成 $\frac{dT(t)}{dt} + k(T(t) - T_a) = 0$ 的形式），使用 `sympy.diff` 或 `diff` 方法对 $T(t)$ 表达式进行求导

   ```python
   ode = T(t).diff(t) + k * (T(t) - Ta)  # 定义 ode 表达式
   print(ode)  # k*(-T_a + T(t)) + Derivative(T(t), t)
   ```

3.  `sympy.dsolve` 求解 ODE 的一般解

   ```python
   ode_sol = sympy.dsolve(ode)  # 求解 ode
   print(ode_sol)  # Eq(T(t), C1*exp(-k*t) + T_a)
   ```

4. 利用初始条件求解未知的积分常数

   ```python
   print(ode_sol.lhs)  # 求解 ode 的 ode_sol 的左侧 T(t)
   print(ode_sol.rhs)  # 求解 ode 的 ode_sol 的右侧 C1*exp(-k*t) + T_a
   ics = {T(0): T0}  # 初始条件
   C_eq = sympy.Eq(ode_sol.lhs.subs(t, 0).subs(ics), ode_sol.rhs.subs(t, 0))
   print(C_eq)  # Eq(T_0, C1 + T_a)
   C_sol = sympy.solve(C_eq)  # 求解未知的积分常数
   print(C_sol)  # [{C1: T_0 - T_a}]
   ```

5. 带入求得的积分常数至 ODE 的一般解中

   ```python
   ode_solve = ode_sol.subs(C_sol[0])
   print(ode_solve)  # Eq(T(t), T_a + (T_0 - T_a)*exp(-k*t))
   ```

以上内容可以归纳成函数

```python
def apply_ics(sol, ics, x, known_params):
    """
    Apply the initial conditions (ics), given as a dictionary on
    the form ics = {y(0): y0, y(x).diff(x).subs(x, 0): yp0, ...},
    to the solution of the ODE with independent variable x.
    The undetermined integration constants C1, C2, ... are extracted
    from the free symbols of the ODE solution, excluding symbols in
    the known_params list.
    """

    free_params = sol.free_symbols - set(known_params)
    eqs = [(sol.lhs.diff(x, n) - sol.rhs.diff(x, n)).subs(x, 0).subs(ics)
           for n in range(len(ics))]
    sol_params = sympy.solve(eqs, free_params)

    return sol.subs(sol_params)
```

```python
print(ode_sol)  # Eq(T(t), C1*exp(-k*t) + T_a)
ode_solve = apply_ics(ode_sol, ics, t, [k, Ta])
print(ode_solve)  # Eq(T(t), T_a + (T_0 - T_a)*exp(-k*t))
```

例二

二阶 ODE：阻尼谐振子， $\frac{d^2x(t)}{dx^2} + 2\gamma \omega_0 \frac{dx(t)}{dt} + \omega_0^2x(t) = 0$ ，其中 $x(t)$ 是振荡器在 $t$ 时刻的位置，$\omega_0$ 是无阻尼情况的频率，$\gamma$ 是阻尼比，初始条件 $x(0) = 1, \quad \frac{dx(t)}{dt}\Bigr|_{t=0} = 0$ 振荡器对任意 $t, \quad \omega_0, \quad \gamma$ 的动力学解。然而，将对应于临界阻尼的 $\gamma = 1$ 直接代入该表达式会导致 `division by zero` 错误，对于 $\gamma$ 的这种特殊选择，需要小心计算 $\gamma \to 1$ 时的极限。

```python
t, omega0, gamma = sympy.symbols("t, omega_0, gamma", positive=True)  # 定义符号
x = sympy.Function('x')  # 定义函数
ode = x(t).diff(t, 2) + 2 * gamma * omega0 * x(t).diff(t) + omega0 ** 2 * x(t)  # 定义 ode 表达式

ode_sol = sympy.dsolve(ode)  # 求解 ODE 的一般解
print(ode_sol)

ics = {x(0): 1, x(t).diff(t).subs(t, 0): 0}  # 利用初始条件求解未知的积分常数
x_t_sol = apply_ics(ode_sol, ics, t, [omega0, gamma])
print(x_t_sol)
```
